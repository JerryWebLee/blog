## 🔧 自定义配置

### 添加新文章

在 `app/pages/blog/index.vue` 中的 `articles` 数组中添加：

```typescript
{
  id: 7,
  title: '新文章标题',
  slug: 'new-article-slug',
  excerpt: '文章摘要...',
  content: '文章内容...',
  coverImage: '封面图片URL',
  publishDate: '2024-01-20',
  readTime: 10,
  category: 'tech',
  tags: ['标签1', '标签2'],
  author: {
    name: '作者名',
    avatar: '头像URL'
  }
}
```

### 修改样式

- 主要样式在组件的 `<style>` 部分
- 全局样式可以修改 `app.vue`
- Tailwind 配置在 `tailwind.config.js` 中
